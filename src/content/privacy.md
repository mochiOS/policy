# プライバシーポリシー

mochiOS関連サービスでは、サービスの提供、保守、セキュリティ確保のため、必要な範囲で利用者に関する情報を取り扱います。

## 取得する情報

当サービスでは、次の情報を取得する場合があります。

- GitHub OAuthで取得するGitHub ID、ユーザー名、表示名、アバターURL
- 開発者ID、ログイン状態、セッションCookie
- 登録された公開鍵、Bundle ID、アプリ情報、リリース情報
- アプリパッケージ、署名、証明書申請に関する情報
- IPアドレス、User-Agent、アクセス日時、操作ログ
- 問い合わせ時に送信された内容

## 利用目的

取得した情報は、次の目的で利用します。

- ログイン、本人確認、開発者アカウント管理
- アプリ登録、審査、配信、証明書発行
- 不正利用、権利侵害、セキュリティ事故の防止
- サービスの保守、改善、障害対応
- 利用者からの問い合わせ対応

## 第三者提供

法令に基づく場合、または不正利用やセキュリティ対応に必要な場合を除き、取得した情報を第三者に提供しません。

## 外部サービス

当サービスでは、ログイン機能としてGitHub OAuthを使用します。GitHub上で認可された範囲の情報のみを取得します。

## Cookie

当サービスでは、ログイン状態の維持やセキュリティのためにCookieを使用します。

## 情報の削除

利用者は、運営者に対してアカウント情報や関連データの削除を依頼できます。ただし、法令上またはセキュリティ上必要な情報は一定期間保持する場合があります。

## 問い合わせ

プライバシーに関する問い合わせは、mochiOS運営者まで連絡してください。

- メール:
    security@mochios.org

## 免責事項

当サービスは、利用者の情報を適切に管理するよう努めますが、情報漏洩や不正アクセスなどのセキュリティ事故が発生した場合の責任を一切負いません。利用者は、自己の責任で当サービスを利用するものとします。
